# @ephraimcare/ui

Shared UI components built on shadcn/ui with Tailwind CSS v4.

## Contents

- `components/` — shadcn/ui components (Button, Card, Dialog, Input, Select, Table, etc.)
- `styles/globals.css` — Global CSS with Tailwind imports and CSS variables
- Brand colors: primary green (#66BB6A), accent teal (#00BFA5)

## Usage

```typescript
import { Button, Card, Input } from '@ephraimcare/ui'
```

Import global styles in your app's root layout:

```typescript
import '@ephraimcare/ui/styles/globals.css'
```
